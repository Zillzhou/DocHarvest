# 更新日志

所有重要的项目更改都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.0.0] - 2024-10-16

### 新增
- ✨ 首次发布
- 🎨 基于 PyQt5 的图形用户界面
- 🔐 飞书 OAuth 2.0 认证支持
- 📥 支持获取飞书文档内容
- 📝 Markdown 格式导出
- 📊 实时进度显示
- 📋 完整的日志系统
- 💾 自定义保存路径
- 🔧 配置文件管理（config.json / config_local.json）
- 📦 PyInstaller 打包支持

### 支持的功能
- 新版文档（/docx/）爬取
- 旧版文档（/docs/）爬取
- 知识库（/wiki/）爬取
- 自动提取文档 ID
- 文档标题获取
- 纯文本内容导出
- 智能标题识别

### 技术栈
- Python 3.9+
- PyQt5 5.15.9
- requests 2.31.0
- PyInstaller 6.3.0

## 已知问题

### v1.0.0
- Markdown 格式识别不够精确（依赖纯文本 API）
- 不支持富文本格式（加粗、斜体等）
- 不支持图片和附件下载
- 不支持表格结构
- 批量导出需要手动逐个操作
